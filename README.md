## 🕹️ Управление (клавиатура)

| Кнопка | Действие |
|--------|----------|
| **WASD** или **Стрелки** | Ходить (вверх/вниз/влево/вправо) |
| **J** или **Z** | Удар (нажимай быстро!) |
| **K** или **X** | Спецудар (бьёт всех рядом, но тратит здоровье!) |
| **L** или **C** | Подобрать оружие с земли |
| **ESC** | Пауза |
| **R** | Рестарт уровня |
| **Enter** | Начать игру / подтвердить |

### На телефоне (Chrome):
На экране появятся виртуальные кнопки (крестовина + кнопки A, S, P)

### Уровни:

1. 🌙 **Ночная улица** — первое знакомство с гопниками и нариками
2. ❄️ **Зимний снегопад** — метель, холод, ещё больше отморозков
3. 🏭 **Промзона** — заброшенные заводы, опасная территория
5. 🌳 **Ночной парк** — темно и страшно
6. 🏢 **Полицейский участок** — побег из ментовки
7. 🏠 **Подъезд** — последний рывок до квартиры!

## 💡 Советы

- **Оружие**: Подбирай ножи и биты — они наносят больше урона
- **Магомед**: Не трогай его если не готов к бою с четверыми!
- **Мент**: Добивай быстро, пока не достал пистолет
- **Спецудар**: Используй когда окружили — бьёт всех вокруг, но стоит 20 HP
- **Нарик**: Следи за спиной — он подкрадывается!

## 🐛 Проблемы?

| Проблема | Решение |
|----------|---------|
| Чёрный экран | Открой консоль (F12) и посмотри ошибки |
| Тормозит | Попробуй Chrome, он самый быстрый для игр |
| Не работает управление | Кликни по игре чтобы она получила фокус |
| Мелкое изображение | Игра автоматически масштабируется под окно |
